Diabetes Prediction System using Flask

Project Overview

The Diabetes Prediction System is a Machine Learning web application developed using the Flask framework. The system predicts whether a person is likely to have diabetes based on medical input parameters such as glucose level, blood pressure, BMI, insulin, age, and other health-related factors.
The application uses a trained Machine Learning model to analyze the user inputs and provide prediction results through a simple and user-friendly web interface.

Features

User-friendly web interface using HTML and CSS
Diabetes prediction using Machine Learning
Data preprocessing and model training
Prediction based on user medical inputs
Flask-based backend integration
Real-time prediction result display
Responsive and simple UI
